#### 详情

##### 关于充电

泰国的两孔插头和 中国 是一样的，如果你带的都是两孔的充电器（如手机），那就不需要带转换插头。
但如果你有三孔插头的充电器（如电脑），就必须带了，转换插头要选择两扁一圆的，如下图：自查知否有三头的充电器

##### 关于小费

泰国 是习惯收取小费的国家，所以服务都特别周到。一般都是给 20B 纸币，不要给硬币，因为在 泰国，硬币是施舍给乞丐的。
如果没想给酒店打扫人员小费，那就千万不要在房间内放钱，否则打扫的会以为是小费。

##### 退税

在 泰国 只要带有“VAT Refund For Tourists”标志的，都可以办理退税。大多数店铺都要代 泰国 当局加收 7%消费税（VAT）,而入境旅游的外籍人只有在购物后 60 天内携离出境的都可以申请退税优惠的，记得购物时要随身携带护照就行。

退税条件 1.游客（非泰籍的外国入境者）必须持旅游签证从标有 VAT Refund for Tourists 的商店购物，必须在同一天同一家店铺消费不低于 2，000 泰铢，且购物的总额必须超过 5000 泰铢。

2.游客在 泰国 的停留时间不超过 180 天。

3.在指定商店购物时，去相应的客服中心必须领取并填好黄色的 PP10 VAT refund form （退税单，如下图），一般情况下客服人员会帮忙填写并把购物小票一起装订在一起，只要保存等到离境就好。
