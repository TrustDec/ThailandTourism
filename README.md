#### 详情
【关于充电】

泰国的两孔插头和 中国 是一样的，如果你带的都是两孔的充电器（如手机），那就不需要带转换插头。
但如果你有三孔插头的充电器（如电脑），就必须带了，转换插头要选择两扁一圆的，如下图：自查知否有三头的充电器
![avatar][1-1]

![avatar][1-2]
![avatar][1-3]

【关于小费】

泰国 是习惯收取小费的国家，所以服务都特别周到。一般都是给20B纸币，不要给硬币，因为在 泰国，硬币是施舍给乞丐的。
如果没想给酒店打扫人员小费，那就千万不要在房间内放钱，否则打扫的会以为是小费。

【退税】

在 泰国 只要带有“VAT Refund For Tourists”标志的，都可以办理退税。大多数店铺都要代 泰国 当局加收7%消费税（VAT）,而入境旅游的外籍人只有在购物后60天内携离出境的都可以申请退税优惠的，记得购物时要随身携带护照就行。

退税条件 
1.游客（非泰籍的外国入境者）必须持旅游签证从标有VAT Refund for Tourists的商店购物，必须在同一天同一家店铺消费不低于2，000泰铢，且购物的总额必须超过5000泰铢。 

2.游客在 泰国 的停留时间不超过180天。 

3.在指定商店购物时，去相应的客服中心必须领取并填好黄色的PP10 VAT refund form （退税单，如下图），一般情况下客服人员会帮忙填写并把购物小票一起装订在一起，只要保存等到离境就好。

#### 第一天(2018-09-27)

9.27抵达曼谷机场后,预计时间在凌晨2点左右.直接去预定的第一站曼谷酒店--廊曼舒适旅社(贾负责查询廊曼机场到该旅社路线,网上说距离在1.5KM左右),
放置物品,洗漱等休息.
9.27早上11:25的航班飞普吉岛,所以9点左右要到机场.具体时间当点待定!抵达普吉岛后,即刻前往Patong V Hotel,在芭东海滩,贾查询机场到酒店的路线(可能要坐车),
